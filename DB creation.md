# commands

- $ : not in root(admin) user.

- sudo su : to be in root user.

- ls : list all the directory/files.

- pwd : print working directory.

- cd : change directory.

- mkdir : creates directory.

- vi : tool/program (notepad)

- i : insert mode

- :wq : write and quit from current file.
